PyLogic
=======

Scott Torborg - `Cart Logic <http://www.cartlogic.com>`_

For working with Saleae Logic.


Quickstart
==========

Install it::

    $ pip install pylogic

Use it::

    >>> from pylogic.api import API
    >>> api = API()
    >>> api.capture()
